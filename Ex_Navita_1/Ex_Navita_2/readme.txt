O exercício 2 do desafio foi feito seguindo uma forma de desenvolvimento simples porem eficaz, para manipulação de dados foi usado Entity Framework,
e a Web Api foi montada de uma maneira onde existe uma reutilização de código para validação de informações.
Foi implementado a segurança JWT para autenticação de usuário, fornecendo assim maior credibilidade ao sistema e garantindo que não 
exista chance de manipulação por pessoas terceiras.O banco de dados foi Usado SQL Server.
Os arquivos do programa estão separados e agrupados por suas responsabilidades no sistema, e foi usado um arquivo de repositório para os testes de autenticação.
